{# FAQ shortcode: renders a styled FAQ section with Google-compliant FAQPage JSON-LD. Usage in markdown: {% faq() %} [ { "q": "Your question here?", "a": "Your answer here." }, { "q": "Another question?", "a": "Another answer with HTML supported." } ] {% end %} The body must be a valid JSON array of objects with "q" and "a" keys. HTML is allowed in answers for links and formatting. #}
