{# FAQ shortcode: renders a styled FAQ section with Google-compliant FAQPage JSON-LD. Both the visible Q&A and the structured data are rendered server-side at build time, so they are present in the static HTML. This matters because search crawlers and LLM crawlers (GPTBot, ClaudeBot, PerplexityBot, Google-Extended, ...) read the raw HTML and generally do not execute JavaScript. Body must be a valid JSON array of objects with "q" and "a" keys. Answers are rendered as Markdown, so blank lines create paragraphs and inline HTML/links are supported. #} {%- set items = load_data(literal=body, format="json") -%}

{{ item.a | markdown | safe }}
